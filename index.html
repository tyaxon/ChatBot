Asistente Virtual por WhatsApp ChatBot

Este bot automatiza la atención al cliente de una empresa proveedora de internet como RSO, mejorando la eficiencia y reduciendo la carga operativa.

📱 ¿Qué puede hacer el bot?

  • Enviar automáticamente formularios en PDF (suscripción, mudanza, cambio de plan).
  • Atender reclamos técnicos como cortes, lentitud o problemas con el cableado.
  • Mostrar planes de internet disponibles y facilitar el cambio de abono.
  • Solicitar estado del cliente (deuda, plan actual, etc.).
  • Guiar paso a paso a los usuarios con menús interactivos.
💬 Ejemplo: Un cliente escribe "Hola" por WhatsApp, y el bot le pide su correo, valida si es cliente, le envía un código de seguridad (OTP) y luego le muestra un menú para elegir el tipo de trámite o reclamo.

⚙️ ¿Cómo funciona?

  • Desarrollado con Node.js + WhatsApp Web.js.
  • Conectado a base de datos MySQL para verificar usuarios y registrar reclamos.
  • Funciona con Puppeteer para controlar la sesión de WhatsApp.
  • Envío de archivos PDF y mensajes multimedia.

🚀 Beneficios para la Empresa

  • Atención rápida y sin intervención humana para trámites comunes.
  • Disponible 24/7.
  • Reduce la carga de trabajo del personal.
  • Ofrece una experiencia moderna y eficiente a los clientes.
